Windows Python Environment Setup Guide

This repository provides a step-by-step guide for installing and configuring Python, Anaconda, and Jupyter Notebook on Windows.
The guide is designed for beginners and university lab environments to ensure proper setup before starting Python development and data science projects.

Step 1: Install Python
1.	Download the latest version from:
https://www.python.org/downloads/
2.	During installation:

o	Check Install for all users

o	Check Add Python to PATH

o	Click Install Now
4.	Approve installation permissions when prompted.
5.	Disable the Path Length Limit to avoid directory path errors (important if path exceeds 260 characters).
6.	Close the installer once completed.

Step 2: Install Anaconda Navigator
1.	Download from:
https://www.anaconda.com/download
2.	Click Download (Skip Registration if prompted).
3.	Follow installation steps:
o	Click Next
o	Accept License Agreement
o	Select Just Me installation
o	Choose installation directory (avoid spaces & Unicode characters)
o	Select Advanced Options (register as default Python if required)
o	Click Install
4.	Finish installation when complete.

Step 3: Launch Jupyter Notebook
1.	Open Anaconda Navigator
2.	Locate Jupyter Notebook
3.	Click Launch
4.	Select your desired directory
5.	Click New → Python 3
6.	A new .ipynb notebook file will open
